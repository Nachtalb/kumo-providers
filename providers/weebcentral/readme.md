# Weeb Central

- **id:** `weebcentral`
- **base:** https://weebcentral.com
- **langs:** en
- **nsfw:** no

Reference provider for the Kumo registry. Server-rendered HTML plus htmx
fragment endpoints (`HX-Request` headers). Ported from the original
`server/providers/weebcentral.js` adapter (itself derived from keiyoushi).

See [`../../docs/authoring.md`](../../docs/authoring.md) for the fully annotated
walkthrough of this script — it is the worked example.
